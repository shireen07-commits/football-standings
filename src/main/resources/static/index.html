⚽ Football Standings Lookup

🔗 API Endpoints (HATEOAS enabled)

Get Countries:

GET /api/standings/countries

Get Leagues by Country:

GET /api/standings/leagues?countryName=England

Get Teams by League:

GET /api/standings/teams?leagueId=152

Get Team Standing:

GET /api/standings/team?countryName=England&leagueName=Premier%20League&teamName=Manchester%20City

Toggle Offline Mode:

POST /api/standings/offline-mode?enabled=true
